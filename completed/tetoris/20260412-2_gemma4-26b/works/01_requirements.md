# 要件定義書：テトリスゲーム・プログラム修正および機能強化

## 1. プロジェクト概要
本プロジェクトは、Pythonおよび`tkinter`ライブラリを用いたテトリスゲームの機能改善およびバグ修正である。これまでの開発過程で発生した、構文エラー（SyntaxError）、実行時エラー（NameError、TypeError）、およびゲームロジックの不備を解消し、安定して動作するゲームプログラムを完成させることを目的とする。

## 2. 開発・修正の基本方針
1.  **構文の完全なクリーンアップ**: 
    開発過程で混入した、タイポ（誤字）、全角文字の混入、不完全な文字列、重複する引数、不正な代入演算子（walrus operatorの誤用など）を全て排除する。
2.  **ロジックの整合性確立**: 
    「ゲームオーバー判定」「ライン消去時のUI更新」「再開機能」が、エラーなく一連の流れとして動作するように実装する。
3.  **UI/UXの最適化**: 
    ユーザーへの指示（Restartメッセージ等）が適切な位置に表示されるよう、レイアウト設計を再定義する。

## 3. 機能要件

### 3.1 ゲームロジック
*   **Game Over 判定の実装**:
    *   新しいブロック（Piece）がスポーンした際、フィールドの上部（最上段）に衝突、または既にブロックが積み上がっている状態で、移動・回転が不可能な状態になった瞬間、即座に `Game Over` 状態へ遷移すること。
*   **ライン消去処理**:
    *   横一行のブロックが揃った際、そのラインを消去し、上部のブロックを一段下げること。
    *   ライン消去直後のスコアおよびレベルの更新が、エラーなく実行されること。
*   **ゲームリセット機能**:
    *   `R` キーの入力により、現在のゲーム状態（スコア、レベル、フィールド、配置済みブロック）を初期化し、即座に再開できること。

### 3.2 ユーザーインターフェース (UI)
*   **Game Over 表示**:
    *   ゲームオーバー時は、ブロックフィールド（Play Field）の**中央にオーバーラップ（重なる形）**して「Game Over」というテキストを表示すること。
    *   「Press R to Restart」というメッセージは、上記「Game Over」のテキストの**直下**に表示すること。
*   **ステータス表示（Score/Level）**:
    *   ScoreおよびLevelの表示エリアは、ブロックフィールドの**枠外上部**に配置すること。
*   **サイドバー/情報表示**:
    *   サイドバー内のラベル（Info Label）の属性（`bg` 等）に重複や不正な値（`#2int` 等）が含まれないよう、正しいカラーコード（例: `#222`）を適用すること。

## 4. 非機能要件（技術的修正事項）

### 4.1 構文・実行時エラーの修正リスト（技術者への指示）
以下のエラー箇所を特定し、修正すること。

*   **SyntaxError (構文エラー) の解消**:
    *   `self.get_new Monte_Carlo_Piece()` → 正しいメソッド名（例: `self.get_new_piece()`）へ修正。
    *   `self.cols * self.・・cell_size` → 全角文字（`・`）を削除し、正しい変数名へ修正。
    *   `self.rows = 2ta_rows = 20` → 不正な数値リテラルを削除し、`self.rows = 20` 等の正しい代入へ修正。
    *   `'L': ([[0, 0, 1], [1, 1, 1]], "#FFA50` → 閉じられていない文字列（Unterminated string literal）を修正。
    *   `self.root leaves_root = root` → 不正な代入構造を修正し、`self.root = root` 等へ整理。
    *   `for r in range(self.grid_empty_check := self.grid_height):` → Walrus operatorの構文エラー、または不要な代入を修正。
    *   `self.top:top_frame (context) ...` → 不正なコロン・マッピング・構文を修正。
    *   `self.root import tk` → `import tkinter as tk` 等、正しいインポート文へ修正。
*   **NameError (定義未定義エラー) の解消**:
    *   `self.check_collision_cells` 内の `nb` → `ny`（または適切な変数名）へ修正。
    *   `self.update_labels` 内の `selfint_score(self.score)` → `self.format_score(self.score)` 等、存在するメソッド名へ修正。
*   **Keyword Argument Error (引数重複) の解消**:
    *   `tk.Label` 内の `bg` 引数の重複（`bg="#2int", ..., bg="#222"`）を解消し、単一の正しい値に統合すること。

## 5. 成果物
*   Pythonソースコード（`program.py`）
    *   エラーなく実行可能であること。
    *   `R`キーによるリスタートが機能すること。
    *   ゲームオーバー表示が指示通り（フィールド内オーバーラップ）であること。
